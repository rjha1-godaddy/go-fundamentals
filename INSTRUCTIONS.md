# Installation and First Program

## Download Installer

<https://go.dev/dl>

## Create a module

```bash
go mod init github.com/rjha1-godaddy/go-fundamentals
```

### Generated go.mod

```bash
module github.com/rjha1-godaddy/go-fundamentals

go 1.24.0
```

## Build and run

```bash
go build main.go

./main
```

## Run without building

```bash
go run main.go
```
